Twitter mood project for course 02806 Social data analysis and visualization F15

Content of website

Mood Map
 A map showing how the mood in an area is, based on tweets from the period: 13/04/2015 - 21/04/2015.
Tweet mood classifier
 Classifies tweets as happy/unhappy based on user input. This model is based on a Multinomial Naive Bayes classifier and trained on happy/unhappy tweets from the period: 13/04/2015 - 21/04/2015 where tweets containing a happy smiley are classified as happy and tweets containing an unhappy smiley are classified as unhappy.
Tweet search/trend map
 A map of New York allowing you to search and locate tweets by the content. Also trends gathered from the Twitter Api is shown and searchable.
Weekly events
 A list of tweets containing information about events for different days of the week.
